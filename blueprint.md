# Lotto Number Generator

## Overview

This is a simple web application that generates and displays random lottery numbers. It's built with modern HTML, CSS, and JavaScript, and it uses Web Components to display the numbers.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: The main stylesheet.
*   `main.js`: The main JavaScript file, containing the logic for generating numbers and the Web Component definition.
*   `blueprint.md`: This file, documenting the project.

## Features

*   Generates 6 unique random numbers between 1 and 45.
*   Displays the numbers in a visually appealing way.
*   A button to generate a new set of numbers.
*   Modern and responsive design.
